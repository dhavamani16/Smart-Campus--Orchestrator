# SmartCampus Orchestrator

**SmartCampus Orchestrator** is an AI-powered multi-agent system designed to transform chaotic campus events into structured, personalized experiences. It intelligently recommends the best technical events for students and guides them to the correct venue.

---

##  Features

*  **Personalized Event Recommendation**

  * Suggests events based on user interests (AI, Coding, Robotics)

*  **Multi-Agent System**

  * User Agent → Captures preferences
  * Recommendation Agent → Selects best events
  * Schedule Agent → Avoids time conflicts
  * Navigation Agent → Provides venue guidance

*  **Smart Scheduling**

  * Automatically filters overlapping events

*  **Venue Navigation**

  * Displays event location and directions

*  **Gaming-Style UI**

  * Dark theme with neon glow
  * Interactive dashboard layout

*  **Real-Time Interaction**

  * Instant recommendations based on user input

##  How It Works

1. User selects interests and availability
2. System processes input using multiple agents
3. Events are filtered and ranked
4. Conflicts are removed
5. Best events are displayed with venue details
